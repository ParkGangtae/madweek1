# MadTicket
"몰입캠프 영화상영회" 예매하고 리뷰 작성까지!

## Outline
MadTicket은 몰입캠프 영화 상영회에서 보일 20개의 영화를 함께 보고 싶은 2분반 구성원과 예매를 하고 후기도 작성할 수 있는 앱입니다.

세 개의 탭은 각각 Members, Movie Gallery, Review로 구성되어 있습니다.

**개발환경**
- Android Studio (Java)
- Adobe Illustrator, Adobe Photoshop
